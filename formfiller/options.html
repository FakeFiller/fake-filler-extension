Field Type Settings

Form Filler uses the ID, NAME and CLASS values of an input field to determine its data type. You can adjust the search phrases in the text boxes below to tweak how Form Filler determines the correct data types.

Basically, any partial match in the attributes underlined:
<input type="anything" id="match" name="match" class="match" />


For example: <input type="text" name="confirmpassword" />
